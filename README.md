# 余烬编年 · Ember Chronicle

一款 **AI 驱动的日系奇幻文字冒险**：你将以“校史者”的身份，在人类与魔族叙事之间做出抉择，
沿着余烬、记忆与真相，走向属于你的终章。

[🎮 立即游玩（Vercel）](https://story.vercel.app)
</div>

---

## ✨ 这款游戏有什么特别？

- **“轻小说氛围 + 策略卡牌战斗”双线体验**：剧情推进与战斗场景自然切换，不割裂。
- **AI 即时叙事**：你的行动会触发动态文本反馈，形成更个性化的旅程。
- **AI 场景/角色插画**：关键节点有生成式画面辅助沉浸感。
- **阵营分歧叙事**：你可偏向人类侧、魔族侧或保持中立，影响角色关系与结局走向。
- **可重玩性高**：同一章节的选择路径不同，能看到不同同伴、事件与世界观碎片。

---

## 🖼️ 界面与玩法速览

游戏主界面围绕“阅读沉浸 + 决策效率”设计：

- **顶部状态区**：展示阵营、生命值、法力、世界观进度等关键信息。
- **中央剧情区**：承载章节文本、事件描述与 AI 生成内容。
- **选项决策区**：每个选择都会改变后续分支、资源或角色关系。
- **角色/同伴侧栏**：用于查看当前角色状态与旅途中结识的关键人物。
- **战斗界面**：采用轻量卡牌指令（输出、防御、回复、特殊技）进行回合制对抗。

> 推荐体验方式：
> 戴耳机 + 全屏 + 慢节奏阅读，会更有“余烬感”。

## 🚀 本地运行

### 1) 环境要求

- Node.js 18+
- npm 9+

### 2) 安装依赖

```bash
npm install
```

### 3) 配置密钥

在项目根目录创建 `.env.local`，并写入：

```bash
GEMINI_API_KEY=your_api_key_here
```

### 4) 启动开发环境

```bash
npm run dev
```

默认访问：<http://localhost:3000>

## 🧭 开发脚本
